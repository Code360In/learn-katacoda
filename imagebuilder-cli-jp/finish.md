# このラボでは以下を実施しました:
* マシンイメージの作成に使用するためのカスタム設計図を作成
* カスタム Red Hat Enterprise Linux マシンイメージの構築
* イメージのビルドプロセスを監視
* 完成したマシンイメージの検証
